**Engine     :** Unreal 5.6.1

**Tools        :** Perforce (Version Control)

**Team size :** 10 full time (1 producer, 2 game designer, 2 engineers, 5 artists)

## My Responsibilities
I was one of [two](https://github.com/Artur-92) engineers working on Frosty Fire. I was primarily responsible for gameplay related systems, like the gameplay mechanics and the in game UI, as well as keeping an eye on the architecture and ensure we make use of the unreal architecture.

My main contributions include:
- Building and structured **network relaveant gameplay systems**
- Ensure proper replication for UI relevant information
- Developing and implementing a HUD system, that is re-useable for the different game modes
- Implemented different game modes
